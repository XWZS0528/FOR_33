## 后台管理

**后台地址** → [lisa1021.com/admin](https://lisa1021.com/admin/)

用你的 GitHub 账号 `huhu-1021` 登录，就可以：

- 编辑你的头像、名字、简介
- 新建相册、上传照片
- 给每组照片写标题和描述

### 新建相册步骤

1. 打开后台 → 点「相册」→ 点「New 相册」
2. 填写年份、标题、描述
3. 点「Add photos」上传照片
4. 右上角点「Publish」保存

照片上传后网站会自动更新，通常一两分钟内生效。
